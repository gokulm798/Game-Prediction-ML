# 🎮 Game Analytics & Player Engagement Prediction

## 📌 Problem
Understanding factors that influence game popularity and player engagement.

## 🎯 Objective
Analyse game data and build ML models to predict engagement and identify key drivers.

## 🛠 Tools
Python, Pandas, Scikit-learn, Matplotlib

## 🔍 Work Done
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Built Decision Tree and Regression models  

## 📈 Key Insights
- Moderate complexity games show higher engagement  
- Playtime strongly influences popularity  
- Age category impacts game adoption  

## 💡 Impact
Supports data-driven game design and marketing decisions.
